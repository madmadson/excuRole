# PROJECT.md

## Project Name

Expo Starter App

---

## Vision

This app is developed iteratively using Claude Code.

Goal:
Build a mobile app step-by-step with a stable and scalable structure.

---

## Initial Goal

Create a minimal app with:

- Home screen
- Second screen
- Navigation
- Simple interaction

---

## Principles

- always runnable
- small steps
- no overengineering
- clarity over complexity

---

## Non Goals (for now)

- no backend
- no authentication
- no complex state management

---

## Success Criteria

- app starts reliably
- features can be added incrementally
- structure remains understandable
