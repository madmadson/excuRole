# Tasks

This folder contains all implementation tasks for the project.

## Rules

- Each file = exactly ONE task
- Tasks must be small and executable
- Tasks must not overlap
- Tasks should be ordered numerically

## Workflow

1. Pick next task
2. Ask Claude to implement it
3. Run smoke test
4. Mark task as done
5. Move to next task

## Execution command (for Claude)

Use:

Read CLAUDE.md, docs/* and tasks/<task-file>.md.
Implement the task within constraints.
Then summarize changes and explain how to test.
